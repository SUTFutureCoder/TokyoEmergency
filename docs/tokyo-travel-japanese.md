# 东京单人旅行日语高频词汇与句子

> 参考“麦克美伦红词 / 星级词汇”的思路：优先掌握高频、可复用、能救场的词和句子。  
> 标记：★★★ 必须会；★★ 高频常用；★ 场景词。  
> 读音主要用罗马音，长音用 `ou / ee / aa` 等近似标注，目标是旅行时能开口。

---

## 0. 使用策略

你已经有 N2 基础，东京单人旅行最大的难点通常不是语法，而是：

- 临场听不清
- 服务业语速快
- 新宿、东京、涩谷等大站信息量过载
- 一紧张就不会开口

建议优先练这三类表达：

1. **万能开场**：`すみません`、`お願いします`、`大丈夫です`
2. **场景模板**：`〇〇はどこですか`、`〇〇まで行きたいです`
3. **救命句**：`もう一度お願いします`、`ゆっくりお願いします`、`書いてもらえますか`

## 25. 临场开口原则

完整句当然好，但旅行时更重要的是能让对方立刻理解。紧张时可以直接说关键词：

| 简化说法 | 读音 | 中文意图 |
|---|---|---|
| 新宿駅、どこですか。 | shinjuku-eki, doko desu ka | 新宿站在哪里？ |
| Suica、使えますか。 | suika, tsukaemasu ka | 可以用 Suica 吗？ |
| 免税、できますか。 | menzei, dekimasu ka | 可以免税吗？ |
| この電車、渋谷、行きますか。 | kono densha, shibuya, ikimasu ka | 这班车去涩谷吗？ |
| トイレ、どこですか。 | toire, doko desu ka | 厕所在哪里？ |

最后记住三个救场动作：

1. 听不懂：`もう一度お願いします。`
2. 太快：`ゆっくりお願いします。`
3. 信息复杂：`書いてもらえますか。`
